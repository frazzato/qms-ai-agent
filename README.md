## 🚀 Overview

QMS AI Agent is an intelligent Quality Management System designed for **AS9100** and **ISO 9001** environments.  
It automates document control, internal auditing, and employee training using a modern AI-driven architecture.

This platform transforms traditional QMS workflows into a **smart, interactive, and automated system** that reduces manual effort, improves compliance, and accelerates audit readiness.

---

## 🧠 Core Features

### ✅ **1. Smart Document Master List**
- Auto‑scans `/docs` folder  
- Extracts metadata  
- Displays a controlled document register  
- Tracks sync status and compliance  
- Generates coverage heatmaps  

### 🕵️ **2. AI Audit Engine**
- Clause mapping  
- Nonconformity detection  
- Context‑aware audit responses  
- Real‑time auditor chat interface  

### 🎓 **3. Automated Training Module**
- Generates:
  - 30‑second summaries  
  - Importance explanations  
  - Audit traps  
  - Knowledge‑check questions  
- Validates answers with correct logic  
- Fully interactive UI  

### 📊 **4. Dashboard**
- System health monitoring  
- Last AI sync timestamp  
- Clickable navigation cards  
- Architecture diagram  
- Status indicators
